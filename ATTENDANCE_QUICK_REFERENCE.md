# Quick Reference: Attendance Scanning

## ⚡ Quick Start Checklist
- [ ] Online status: 🟢 Online
- [ ] Location: 📍 Within School Area
- [ ] Time: 🕐 Valid Time (7 AM - 6 PM)
- [ ] Camera permissions granted
- [ ] Students have QR codes ready

## 📱 Scanning Steps
1. Click "📱 Start QR Scan"
2. Point camera at student QR code
3. Wait for green flash + vibration ✅
4. Student marked present automatically

## 🎯 Status Indicators
- 🟢 Green card = Present
- 🟡 Yellow card = Late
- ⚪ Gray card = Absent

## 🔧 Manual Override
- Check/uncheck boxes for Present
- Use dropdown for Absent/Late status

## 💾 Submit Process
1. Review all records
2. Click "💾 Submit Attendance"
3. Data syncs automatically

## 🚨 Emergency Procedures
- **No Camera**: Use manual checkboxes
- **Outside School**: Manual marking (warning recorded)
- **Offline**: Data saves locally, syncs later
- **Time Issues**: Manual marking (validation logged)
