# 📱 Attendance Scanning - Quick Guide

## 🎯 3-Step Process

### 1. CHECK STATUS
```
🟢 Online        📍 School Area    🕐 Valid Time
   ✅               ✅               ✅
```

### 2. SCAN OR MARK
```
📱 SCAN QR: Click → Point → Flash ✅
☑️ MANUAL: Check boxes for Present/Absent
```

### 3. SUBMIT
```
💾 Submit Attendance → Auto-sync
```

## 🚨 Key Warnings

| Warning | Meaning | Action |
|---------|---------|--------|
| 📍 Outside School | >500m from school | Move closer or manual mark |
| 🕐 Outside Hours | Not 7AM-6PM | Manual mark (logged) |
| 🔴 Offline | No internet | Data saves locally |

## 💡 Pro Tips

- **Start Early**: Check validation status before class
- **Good Lighting**: Bright light helps QR scanning
- **Steady Hand**: Hold device 6-12 inches from QR code
- **Backup Ready**: Manual checkboxes always available
- **Regular Submit**: Don't wait - submit attendance often

## 🎨 Status Colors

- 🟢 **Green**: Present students
- 🟡 **Yellow**: Late students
- ⚪ **Gray**: Absent students
- 🔵 **Blue**: Scanning active

## 🔧 Quick Fixes

**Camera not working?**
- Grant permissions
- Refresh page
- Use manual mode

**Location wrong?**
- Enable GPS
- Click "Refresh Location"
- Wait for accurate reading

**Time incorrect?**
- Check device time
- Click "Refresh Time"
- Contact IT if persists

---
**Remember**: The system is designed to be simple, secure, and reliable. Manual marking is always available as backup!
